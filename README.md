## ğŸš€ Getting Started

### Prerequisites

- A GitHub account
- Git installed on your local machine

### Setup Instructions

1. **Create a new repository on GitHub**
   - Repository name should be: `yourusername.github.io` (replace `yourusername` with your GitHub username)
   - Make it public
   - Initialize with a README (optional)

2. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/yourusername.github.io.git
   cd yourusername.github.io
   ```

3. **Add your files**
   - Copy all files from this portfolio project to your repository
   - Add your resume PDF file as `resume.pdf` in the root directory

4. **Enable GitHub Pages**
   - Go to your repository on GitHub
   - Click on "Settings"
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click "Save"

5. **Commit and push your changes**
   ```bash
   git add .
   git commit -m "Initial portfolio commit"
   git push origin main
   ```

6. **Access your site**
   - Your portfolio will be available at `https://yourusername.github.io` within a few minutes
   - It may take up to 10 minutes for changes to propagate
